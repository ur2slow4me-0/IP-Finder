A simpel Java-based tool for finding the IP address of any target URL.
Created as port of my learning journey in cybersecurity and networking fundamentals.

#Features
- Convert URL/Domain into IP Address
- Lightweight and fast
- Easy to use
- Beginner-friendly code

#How it Works
The tool uses Java's built-in InetAddress class to resolve the IP address of a given hostname.

#Usage
1. Clone the repository:
  git clone
  https://github.com/ur2slow4me-0/IP-Finder.git
2. Compile the Java File:
  javac IP-Finder.java
3. Run the Program:
   java IP-Finder
4. Enter your target URL.
